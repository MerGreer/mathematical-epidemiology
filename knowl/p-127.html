We began to explore the differences between the discrete data, collected round-by-round in the Zombie Game App, and the continuous model, which uses a system of differential equations to produce smooth curves. In this chapter, we continue that exploration visually, by producing graphs of data and graphs of model results. We also see a first, very visual, way to estimate the value of \(\beta\) in our model. (There will be other, more detailed, ways to estimate \(\beta\text{,}\) as we continue in this text.) To do all this, we employ visualization tools. This text uses cells that run both Python and SageMath 3 , which “is a free open-source mathematics software system” 4 , for visualization. The Python code here can be used in any setting where you use Python. 6
