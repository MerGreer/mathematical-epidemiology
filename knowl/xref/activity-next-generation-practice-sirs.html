Activity 14.5.

Now that we have shown all the steps for computing a formula for \(\mathcal{R}_0\) in the SEIR model, try again for the SIRS model, shown in Figure 14.1.
Hint.
Only one compartment of the SIR model involves individuals experiencing some stage of infection, namely, the I compartment. This means that when we write the equations in matrix form, we use a \(1 \times 1\) matrix. Then \(\widehat{F}= \begin{pmatrix} \beta SI \end{pmatrix}\) and \(\widehat{V}= \begin{pmatrix} \gamma I \end{pmatrix}.\) Computing partial derivatives with respect to \(I\text{,}\) and evaluating at the disease-free equilibrium \((S, I, R)=(N, 0, 0)\text{,}\) results in \(F = \begin{pmatrix} \beta N \end{pmatrix}\) and \(V = \begin{pmatrix} \gamma \end{pmatrix}.\) Computing \(FV^{-1}\) results in \(FV^{-1} = \begin{pmatrix} \frac{\beta N}{\gamma} \end{pmatrix}\text{,}\) and the one eigenvalue of \(FV^{-1}\) tells us that \(\mathcal{R}_0 = \frac{\beta N}{\gamma}\text{.}\)
