Activity 8.4.

Each of the following changes a statement from Activity 8.2. Based on the change, comment on what we should expect regarding the statement’s conclusion.
  1. Recall the statement “Suppose a function \(f\) is continuous for \(x\)-values satisfying \(a \leq x \leq b\text{.}\) In this case, \(f\) takes on every value between the numbers \(f(a)\) and \(f(b)\)”. Suppose a function \(g\) is not continuous at some of the \(x\)-values satisfying \(a \leq x \leq b\text{.}\) In this case, might there be some numbers between \(f(a)\) and \(f(b)\) for which \(f\) does not take on the value of those numbers?
  2. Recall the statement “Differentiable functions are continuous”. Suppose a function is differentiable at all points except at \(x=0\text{.}\) Must this function be continuous everywhere? Explain in terms of mathematical logic, and if possible, provide one or more examples of functions that demonstrate that when an assumption is not satisfied, then the conclusion may not be satisfied.
Hint.
When any of the assumptions are not satisfied, we should not expect the conclusion to be satisfied.
