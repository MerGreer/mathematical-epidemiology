Activity 7.4.

Each of the following changes a statement from Activity 7.2. Based on the change, comment on what we should expect regarding the statement's conclusion.
  1. Recall the statement “If a polygon has six sides and six angles, then the polygon is a hexagon”. Suppose you see a polygon that has seven sides and seven angles. Must this polygon be a hexagon?
  2. Recall the statement “Suppose a function \(f\) is continuous for \(x\)-values satisfying \(a \leq x \leq b\text{.}\) In this case, \(f\) takes on every value between the numbers \(f(a)\) and \(f(b)\)”. Suppose a function \(g\) is not continuous at some of the \(x\)-values satisfying \(a \leq x \leq b\text{.}\) In this case, might there be some numbers between \(f(a)\) and \(f(b)\) for which \(f\) does not take on the value of those numbers?
