The time range for this numerical solution appears in line 13, using the arange command from the numpy package downloaded in line 2. The code says t = np.arange(0.0, 8.0, 0.1). This means that time starts at \(t=0.0\) and continues till \(t=8.0\text{.}\) In addition, the 0.1 tells Python to use a step size of 0.1 in the “step through” process for computing \(S(t)\) and \(I(t)\) values.
