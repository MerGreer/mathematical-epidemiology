Recall the statement “Suppose a function \(f\) is continuous for \(x\)-values satisfying \(a \leq x \leq b\text{.}\) In this case, \(f\) takes on every value between the numbers \(f(a)\) and \(f(b)\)” from Activity 7.2. Suppose a function \(g\) is not continuous at some of the \(x\)-values satisfying \(a \leq x \leq b\text{.}\) In this case, might there be some numbers between \(f(a)\) and \(f(b)\) for which \(f\) does not take on the value of those numbers?
